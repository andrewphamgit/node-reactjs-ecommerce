# React + Vite

+ Create-project:
    1. Create: `npm create vite@6.2.0` -> reactjs -> javascript
    2. Install:
        `npm install axios react-router-dom react-toastify`
         - Tailwindcss:
          `npm install -D tailwindcss@3 postcss autoprefixer`
          `npx tailwindcss init -p`


+ Run project: `npm run dev`
